# BookTier - Favourite Book Lists

## Using:
- Svelte & Svelt-kit
- Typescript
- Firebase (Auth, Storage, Hosting)
- Tailwind & DaisyUI

## Features:
- Google OAuth
- Unique profile names
- List & drag-n-drop favorite books

## Instructions:
- $ npm i && npm run dev
- $ firebase deploy
